## ✅ 已實作的改進

### 1️⃣ 新增換手率特徵 (Turnover Rate)
**權重: 15%** ⭐

**原理:**
```python
換手率 = 當前成交量 / 流通股數 (估算)
```

**為什麼重要？**
- 主力進場通常伴隨高換手率
- 換手率 > 5% 代表活躍交易
- 換手率 > 10% 代表超高活躍度，主力可能性高

**標準化:**
- 5% 換手率 = 正常
- 10% 換手率 = 高 (得分 1.0)

---

### 2️⃣ 改進大單檢測邏輯 (標準差方法)

**舊方法問題:**
```python
大單閾值 = 平均量 × 5  # 太粗糙
```

**新方法 (統計學方法):**
```python
大單閾值 = 平均量 + 2.5 × 標準差  # 99% 信賴區間
```

**額外改進:**
- ✅ 偵測連續大單 (間隔不超過 2 天)
- ✅ 連續 3 筆以上大單額外加分 +0.3
- ✅ 更科學、更準確

**範例:**
```
股票 A 成交量歷史: [100, 120, 110, 105, 500]
平均: 187
標準差: 176
大單閾值: 187 + 2.5×176 = 627

判斷: 最後一筆 500 < 627，非大單
但如果是 650，則為大單 ✅
```
