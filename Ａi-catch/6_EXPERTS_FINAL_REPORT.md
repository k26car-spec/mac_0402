## 📊 新增专家详解

### 5. 趋势识别专家 ✅

**核心功能**:
- 均线趋势（多头/空头排列）
- 价格趋势（上升/下降/盘整）
- 趋势强度评估
- 趋势一致性确认

**检测逻辑**:
```python
# 多头排列
if ma5 > ma20 > ma60:
    趋势强度 = (ma5 - ma60) / ma60
    → 买入信号

# 空头排列  
if ma5 < ma20 < ma60:
    趋势强度 = (ma60 - ma5) / ma60
    → 卖出信号

# 趋势确认
价格方向 = 价格变化
均线方向 = (ma5 - ma20) / ma20
if 方向一致 → 确认趋势
```

**输出示例**:
```
"多头排列，趋势强度13%；温和下跌；价格与均线方向一致"
```

### 6. 支撑阻力专家 ✅

**核心功能**:
- 52周高低点分析
- MA20/MA60动态支撑阻力
- 整数关口（心理价位）
- 突破/跌破信号

**检测逻辑**:
```python
# 相对位置
position = (当前价 - 52周最低) / (52周最高 - 52周最低)

if position > 0.9:
    → 接近阻力，警惕
elif position < 0.1:
    → 接近支撑，关注反弹

# 整数关口
if price ≈ 整数(100/50):
    → 心理价位压力/支撑

# 突破信号
if 放量 and 涨幅 > 3%:
    → 突破确认
```

**输出示例**:
```
"接近52周最高(92%位置)；MA20支撑测试；整数关口500压力"
```
