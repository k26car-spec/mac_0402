### 2. 智能潛力評分

**評分維度**:
- 趨勢強度（40分）- 週/月/雙月成長率
- 趨勢一致性（20分）- 多週期同步性

**特殊處理**:
- 高價股（>300元）成交量標準降低
- 短線過熱會減分
- 多週期向上加分

## 🎯 使用場景

### 場景 1: 尋找高潛力股

**步驟**:
1. 查看「潛力評分」欄位
2. 關注評分 70+ 的股票
3. 查看「關鍵信號」確認多週期向上
4. 檢查「分析」了解詳情

**範例**:
- 3563 牧德：評分 75，關鍵信號「✓ 長線看漲」
- 3380 明泰：評分 72，關鍵信號「✓ 成交活絡」

### 場景 2: 識別風險股票

**步驟**:
1. 查看評分 < 50 的股票
2. 注意「✗」負面信號
3. 閱讀「分析」了解趨勢

**範例**:
- 3450 聯鈞：評分 45，「✗ 長線走空」
- 3518 柏騰：評分 38，「✗ 月線下跌」

### 場景 3: 判斷短線過熱

**步驟**:
1. 查看「月成長率」和「週成長率」
2. 如果月成長率遠低於週成長率
3. 注意「⚠️ 短線過熱」信號

**範例**:
- 3466 德晉：月 +0.81% 但雙月只 +0.13%
- 評分雖達 65，但需注意回檔風險

### 場景 4: 發現突發題材

**步驟**:
1. 查看「漲跌幅」> 5% 的股票
2. 對比「週成長率」和「月成長率」
3. 如果週/月成長率接近 0 但今日大漲
4. 可能是突發消息推動

**範例**:
- 3581 博磊：今日 +9.89 元，但週/月成長都是 0
- 需確認題材是否可持續
