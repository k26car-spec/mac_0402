## 💡 **问题根源**

### 归一化阈值错位

**训练过程**:
```python
# 1. Target是未来5天收益率
Target = (Close_future / Close_now - 1) * 100
# 例如：[-5%, -3%, -1%, 0%, +1%, +3%, +5%]

# 2. MinMaxScaler归一化
scaler = MinMaxScaler()
Target_normalized = scaler.fit_transform(Target)
# 映射到[0, 1]

# 关键问题：0%收益映射到哪里？
# 取决于数据分布！
# - 如果数据偏下跌：0% → 0.6左右
# - 如果数据偏上涨：0% → 0.3左右
# - 如果数据平衡：0% → 0.5左右
```

**预测过程**:
```python
# LSTM输出归一化值
prediction = model.predict(X)  # 例如：0.1201

# 错误的判断（使用固定0.5）
if prediction > 0.5:
    decision = "上涨"
else:
    decision = "下跌"  # ← 0.1201 < 0.5，判为下跌

# 但实际上：
# 如果该股票的scaler把0%映射到0.12
# 那么0.1201应该判为"接近0%，略涨"
# 而不是"下跌"！
```

---

### 三支股票的阈值分析

| 股票 | 预测值范围 | 均值 | 数据偏向 | 正确阈值估计 | 使用0.5结果 |
|------|-----------|------|---------|-------------|-----------|
| 2314 | [-0.05, -0.04] | -0.045 | 偏下跌 | **-0.045** | ✅ 碰巧对 |
| 5521 | [0.03, 0.05] | 0.037 | 平衡 | **~0.04** | 🟡 随机 |
| 2408 | [0.11, 0.14] | 0.120 | 偏上涨 | **~0.12** | ❌ 相反 |

**发现**:
- 每支股票的"0收益"对应不同的归一化值
- 使用固定0.5导致系统性错误
- 2314碰巧对（偏下跌数据）
- 2408完全错（偏上涨数据）

---

## ✅ **解决方案**

### 方法1: 反推零点阈值（推荐）⭐

**训练时保存**:
```python
# 在训练脚本中
scaler_y = MinMaxScaler()
Target_normalized = scaler_y.fit_transform(Target)

# 计算0%收益对应的归一化值
zero_threshold = scaler_y.transform([[0]])[0][0]

# 保存scaler和阈值
import pickle
with open(f'models/lstm_smart_entry/{stock}_scaler.pkl', 'wb') as f:
    pickle.dump({
        'scaler': scaler_y,
        'zero_threshold': zero_threshold
    }, f)
```

**预测时使用**:
```python
# 加载阈值
with open(f'models/lstm_smart_entry/{stock}_scaler.pkl', 'rb') as f:
    info = pickle.load(f)
    threshold = info['zero_threshold']

# 使用正确阈值
prediction = model.predict(X)
decision = 1 if prediction > threshold else 0
```

**优点**: 精确，理论正确  
**缺点**: 需要重新训练或有scaler文件

### 方法2: 验证集最佳阈值

**实现**:
```python
def find_optimal_threshold(model, X_val, y_val):
    """在验证集上找最佳阈值"""
    predictions = model.predict(X_val, verbose=0).flatten()
    
    # 测试多个阈值
    thresholds = np.linspace(
        predictions.min(), 
        predictions.max(), 
        100
    )
    
    best_acc = 0
    best_t = 0.5
    
    for t in thresholds:
        pred_bin = (predictions > t).astype(int)
        acc = np.mean(pred_bin == y_val)
        
        if acc > best_acc:
            best_acc = acc
            best_t = t
    
    return best_t
```

**优点**: 不需要重新训练  
**缺点**: 需要验证集，可能过拟合

---

### 方法3: 使用预测值中位数

**实现**:
```python
# 在测试集上
predictions = model.predict(X_test, verbose=0).flatten()
threshold = np.median(predictions)

# 判断
pred_binary = (predictions > threshold).astype(int)
```

**优点**: 最简单，无需额外数据  
**缺点**: 理论不严谨，但实用
