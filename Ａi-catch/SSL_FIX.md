## ❌ 問題說明

系統無法連接 Yahoo Finance 獲取股票數據，錯誤訊息：
```

unable to get local issuer certificate
```

**原因**：Python 3.14 在 macOS 上的 SSL 證書配置問題

## ✅ 已修復

1. ✅ 升級 `certifi` 套件到最新版本
2. ✅ 修改 `async_crawler.py` 使用正確的SSL配置
3. ✅ 所有 HTTP 請求現在使用認證的證書
