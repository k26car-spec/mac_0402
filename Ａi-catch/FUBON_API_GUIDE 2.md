## 🎯 工作原理

### 三層獲取機制

```
查詢股票名稱
    ↓
1. 本地字典 (最快)
    ↓ 找不到
2. Yahoo Finance API (自動獲取)
    ↓ 失敗
3. 富邦 API (備用)
    ↓ 失敗
4. 返回代碼 (後備)
```

### 優點

1. **自動化** - 新股票自動獲取名稱
2. **快取** - 獲取後自動儲存，下次更快
3. **可靠** - 多重備援機制
4. **無痛** - 不影響現有功能

## 🎊 自動快取

### 工作方式

1. **首次查詢**：從 API 獲取（較慢 ~1秒）
2. **自動儲存**：加入 STOCK_NAMES 字典
3. **再次查詢**：從字典讀取（極快 <0.001秒）
4. **持續有效**：24小時快取

### 效能提升

```
第一次查詢 2344.TW: ~1秒   (API)
第二次查詢 2344.TW: <0.001秒 (快取)
```
