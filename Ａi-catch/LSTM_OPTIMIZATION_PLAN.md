### 方案1: 调整预测阈值（最快，2小时）

**如果诊断发现**: 预测值分布偏向某一边

**行动**:
```python
# 修改lstm_backtest.py
# 不用固定0.5，而是动态最佳阈值

# 为每个股票找最佳阈值
best_thresholds = {}
for stock in stocks:
    # 在训练集上找最佳阈值
    best_threshold = find_optimal_threshold(model, X_train, y_train)
    best_thresholds[stock] = best_threshold

# 预测时使用各自最佳阈值
pred_binary = (predictions > best_thresholds[stock]).astype(int)
```

**预期改善**: +5-10%  
**成功率**: 70%

### 方案2: 重新训练with class_weight（中等，3小时）

**如果诊断发现**: 数据不平衡（上涨/下跌比例差异大）

**行动**:
```python
# 修改训练脚本
from sklearn.utils.class_weight import compute_class_weight

# 计算类别权重
class_weights = compute_class_weight(
    'balanced',
    classes=np.unique(y_train_binary),
    y=y_train_binary
)

# h训练时使用
model.fit(
    X_train, y_train,
    class_weight={0: class_weights[0], 1: class_weights[1]},
    ...
)
```

**需要**: 重新训练10-20支表现差的股票
