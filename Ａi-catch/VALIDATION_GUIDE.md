# 📊 验证已改进股票 - 使用指南

## 🎯 目的

验证已改进的19支股票在最新数据下性能是否稳定

---

## 🚀 快速开始

### 基础验证（推荐）

```bash
cd /Users/Mac/Documents/ETF/AI/Ａi-catch

# 验证所有已改进的股票
python3 validate_improved_stocks.py

# 预计时间: 15-30分钟
```

---

## 📊 脚本功能

### 自动执行步骤

1. **加载已改进结果**
   - Top 4结果
   - 批量处理结果
   - 剩余3支结果
   - **总计19支**

2. **获取最新数据**
   - 每支股票365天历史数据
   - 使用yfinance自动获取

3. **重新评估性能**
   - 使用与改进时相同的方法
   - Augmented或Regularized
   - 相同的配置参数

4. **对比分析**
   - 原始MAE vs 当前MAE
   - 计算变化百分比
   - 判断稳定性

5. **生成报告**
   - 详细验证结果
   - 需要重训列表
   - 统计摘要

## 🎯 性能判断标准

| 变化幅度 | 状态 | 说明 |
|---------|------|------|
| ≤ -5% | 📈 提升 | 性能改善，非常好 |
| -5% ~ +5% | ✅ 稳定 | 性能稳定，正常 |
| > +5% | ⚠️ 下降 | 需要重新训练 |

**为什么5%？**
- 考虑数据的自然波动
- 防止过度敏感
- 平衡效率和质量

### 问题2: 数据获取失败

```
[5/19] 2337
  原始: MAE=0.1434 (Augmented)
  获取最新数据... 失败

**原因**: 网络问题或股票代码错误

**解决**:
- 检查网络连接
- 确认股票是否下市
- 稍后重试

### 问题3: 评估失败

```

**原因**: 模型训练错误

**解决**:

- 检查错误日志
