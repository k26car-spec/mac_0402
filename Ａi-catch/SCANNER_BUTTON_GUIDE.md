# 🎯 Streamlit 儀表板 - 全市場掃描按鈕使用指南

## ✨ 新功能：一鍵掃描全市場

現在您可以直接在 Streamlit 儀表板上點擊按鈕掃描全市場，找出績優股！

## 📍 功能位置

**側邊欄** → **🔍 全市場掃描** 區塊

## 🎮 使用步驟

### 步驟 1: 開啟儀表板

```bash
cd /Users/Mac/Documents/ETF/AI/Ａi-catch
./start_backend.sh
```

訪問: **http://localhost:8501**

### 步驟 2: 設定掃描條件

在側邊欄 **🔍 全市場掃描** 區塊中：

#### 📊 最低潛力評分（滑桿）
- **拖動滑桿**設定最低評分門檻
- 預設值：70 分
- 建議設定：
  - 嚴格篩選：80-90 分
  - 平衡篩選：70 分（預設）
  - 寬鬆篩選：50-60 分

#### 🔢 掃描數量（下拉選單）
- **點擊選單**選擇要掃描的股票數量
- 選項：10 / 20 / 30 / 50
- 建議設定：
  - 快速測試：10 支
  - 日常使用：20 支（預設）
  - 深度掃描：50 支

### 步驟 3: 點擊「🚀 開始掃描」按鈕

**大型藍色按鈕** - 點擊後：
1. 顯示掃描進度條
2. 即時顯示正在掃描的股票代號
3. 顯示掃描進度 (例如: 掃描中... 2330 (5/20))

### 步驟 4: 查看掃描結果

掃描完成後，側邊欄會顯示：

#### ✅ 成功訊息
```
✅ 找到 8 支績優股！
```

#### 🏆 掃描結果 TOP 5
顯示格式：
```
2330 台積電 (85分)  [➕按鈕]
2454 聯發科 (82分)  [➕按鈕]
3380 明泰   (75分)  [➕按鈕]
...
```

### 步驟 5: 加入監控清單

有兩種方式：

#### 方式 A: 個別加入
- **點擊股票右側的「➕」按鈕**
- 該股票立即加入監控清單
- 頁面自動刷新顯示新股票

#### 方式 B: 全部加入
- **點擊「📥 全部加入監控」按鈕**
- 所有掃描到的績優股一次全部加入
- 顯示加入數量（例如：✅ 已加入 8 支股票）

## 💡 使用技巧

### 技巧 1: 動態調整評分

如果第一次掃描沒找到股票：
1. 降低「最低潛力評分」（例如從 70 降到 60）
2. 再次點擊「🚀 開始掃描」
3. 查看新的結果

### 技巧 2: 分批加入

不確定是否要全部加入時：
1. 先點擊 TOP 3 的「➕」按鈕
2. 觀察這些股票的表現
3. 表現好再加入更多

### 技巧 3: 定期掃描

建議掃描頻率：
- **每日早盤前**: 評分 75 | 數量 20
- **每週一晚上**: 評分 70 | 數量 50
- **每月初**: 評分 60 | 數量 50

### 技巧 4: 搭配篩選使用

掃描後：
1. 在側邊欄勾選「僅顯示高優先度」
2. 或調整「最低成長率」滑桿
3. 進一步篩選掃描結果

## ⚠️ 常見狀況

### ❌ 未找到任何股票

**顯示**: `⚠️ 未找到評分 >= 70 的股票，建議降低評分門檻`

**解決方法**:
1. 將「最低潛力評分」降低（例如從 70 → 60）
2. 增加「掃描數量」（例如從 20 → 50）
3. 再次點擊「🚀 開始掃描」

### ⏳ 掃描時間過長

**原因**: 掃描數量太多或 API 回應慢

**解決方法**:
1. 減少掃描數量
2. 在網路狀況較好時掃描
3. 分批掃描（先掃 20 支，再掃另外 20 支）

### 🔄 頁面卡住

**解決方法**:
1. 等待掃描完成（不要重複點擊按鈕）
2. 如果超過 5 分鐘無反應，刷新頁面
3. 重新設定較小的掃描數量
