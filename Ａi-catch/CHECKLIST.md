## 🚀 快速啟動檢核表

### 首次使用者

- [ ] 1. 確認 Python 3.9+ 已安裝
- [ ] 2. 克隆或下載專案
- [ ] 3. 執行 `pip install -r requirements.txt`
- [ ] 4. 複製 `.env.example` 為 `.env`
- [ ] 5. (可選) 設定通知 Token
- [ ] 6. 執行 `python3 test_system.py` 測試
- [ ] 7. 執行 `./start_monitor.sh` 啟動
- [ ] 8. 查看日誌確認運行

### 進階用戶

- [ ] 1. 編輯 `config.yaml` 自訂監控清單

- [ ] 4. (可選) 訓練自己的 ML 模型
