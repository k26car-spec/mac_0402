## 🔧 如何啟用富邦API真實數據

### 前提條件

1. **安裝富邦Neo SDK**
```bash
pip install fubon-neo
```

2. **安裝額外依賴**
```bash
pip install pydantic-settings
```

3. **配置富邦帳號**
   - 確保 `fubon.env` 文件存在並配置正確
   - 憑證文件 (`.pfx`) 放置在正確位置
   - 帳號、密碼、憑證密碼正確

## 🔍 故障排除

### 問題 1: "No module named 'fubon_client'"
**解決方案**: 
- 確保 `fubon_client.py` 在項目根目錄
- 檢查導入路徑是否正確

### 問題 2: "No module named 'pydantic_settings'"  
**解決方案**:
```bash
pip install pydantic-settings
```

### 問題 3: "富邦API連接失敗"
**解決方案**:
1. 檢查 `fubon.env` 配置
2. 確認憑證文件存在
3. 驗證帳號密碼正確
4. 檢查憑證是否過期

### 問題 4: "技術篩選返回空數組"
**原因**: 富邦API未連接，且備用數據機制未觸發
**解決方案**: 查看後端日誌確認錯誤信息
