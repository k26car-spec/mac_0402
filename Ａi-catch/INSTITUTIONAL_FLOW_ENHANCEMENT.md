## 🎨 新增功能

### **1. 智能判斷買賣超** ✅

```javascript
function formatNetBuy(value) {
    const sign = value > 0 ? '+' : '';
    const color = value > 0 ? 'text-green-600' : 'text-red-600';
    const label = value > 0 ? '買超' : '賣超';
    return `<span class="${color}">${sign}${value}張 (${label})</span>`;
}
```

**特點**:
- ✅ 正數顯示綠色「買超」
- ✅ 負數顯示紅色「賣超」
- ✅ 自動加上+/-符號
- ✅ 明確標示買/賣

---

### **2. 法人一致性分析** ✅

```javascript
function getConsensus(foreign, trust, dealer) {
    const buyCount = [foreign, trust, dealer].filter(v => v > 0).length;
    const sellCount = [foreign, trust, dealer].filter(v => v < 0).length;
    
    if (buyCount === 3) return '📈 三大法人同步買超';
    if (sellCount === 3) return '📉 三大法人同步賣超';
    if (buyCount === 2) return '↗️ 二大法人買超';
    if (sellCount === 2) return '↘️ 二大法人賣超';
    return '⚠️ 法人分歧';
}
```

**判斷邏輯**:
| 情況 | 買超數 | 賣超數 | 標籤 | 顏色 |
|-----|-------|-------|------|------|
| 全買 | 3 | 0 | 📈 三大法人同步買超 | 綠色 |
| 全賣 | 0 | 3 | 📉 三大法人同步賣超 | 紅色 |
| 2買1賣 | 2 | 1 | ↗️ 二大法人買超 | 淺綠 |
| 2賣1買 | 1 | 2 | ↘️ 二大法人賣超 | 淺紅 |
| 其他 | - | - | ⚠️ 法人分歧 | 黃色 |
