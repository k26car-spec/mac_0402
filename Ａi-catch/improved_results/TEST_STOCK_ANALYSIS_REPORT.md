## 🔍 深入分析：為什麼 Baseline 贏了？

### 1. 模型複雜度適配

```
數據複雜度: 簡單 (模擬數據)
Baseline:   簡單架構 ✅ 匹配
Others:     複雜架構 ⚠️ 過度

原則: 簡單數據用簡單模型！
```

### 2. 正則化的權衡

```
Baseline:      無L2正則化 → 訓練MAE: 0.040 ⭐
Regularized:   L2=0.015  → 訓練MAE: 0.094 ❌
Optimized:     L2=0.008  → 訓練MAE: 0.093 ❌
```

**對於簡單數據，正則化反而限制了學習能力！**

### 3. 訓練時間充足

```
Baseline: 113 epochs → 充分學習
Optimized: 34 epochs → 提前停止
```

**Early Stopping 的權衡：**
- ✅ 防止過擬合
- ❌ 可能提前停止優秀模型
