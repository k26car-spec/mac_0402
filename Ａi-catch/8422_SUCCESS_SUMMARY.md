## 💡 学到的教训

### 教训1: 不要使用太小的测试集

```python
# ❌ 不推荐
test_size = 0.15  # 仅15%，容易不稳定

# ✅ 推荐
test_size = 0.25  # 25%更稳定
test_size = 0.20  # 20%也可以
```

### 教训2: 小测试集的风险

**15%测试集的问题**:
- 样本少 → 高方差
- 容易碰到偏难/偏易的时期
- 导致误判

**25%测试集的优势**:
- 样本充足 → 低方差
- 更好的代表性
- 更稳定的评估
