## ✨ 優化內容

### 1. 股票名稱 ✅
**問題**: 顯示 "None" 而不是實際股票名稱

**解決方案**:
```python
# 確保名稱從富邦 API 獲取，如果沒有則顯示代號
stock_name = quote.get('name', '').strip() if quote.get('name') else symbol
# 如果名稱是 "None" 字串或空字串，使用代號
if stock_name in ['', 'None', None]:
    stock_name = symbol
```

**效果**: 
- ✅ 優先顯示富邦 API 返回的股票名稱
- ✅ 如果 API 無名稱，則顯示股票代號
- ✅ 不再出現 "None" 字樣

---

### 2. 現價格式化 ✅
**問題**: 顯示過多小數位（如 `289.000000`）

**解決方案**:
```python
"現價": round(quote['price'], 2),  # 小數點第二位
```

**顯示格式**:
```python
"現價": "{:.2f}"  # 例如: 289.00, 72.20
```

**效果**:
- ✅ 固定顯示 2 位小數
- ✅ 例如：`289.00`、`72.35`

---

### 3. 漲跌幅格式化 ✅
**問題**: 沒有 % 符號，不直觀

**解決方案**:
```python
"漲跌幅": round(quote['change_percent'], 2)
```

**顯示格式**:
```python
"漲跌幅": "{:+.2f}%"  # 帶正負號和 %
```

**效果**:
- ✅ 自動加上正負號：`+2.50%` 或 `-1.30%`
- ✅ 清楚顯示漲跌方向
- ✅ 固定 2 位小數

**範例**:
- 上漲：`+9.89%`
- 下跌：`-0.86%`
- 持平：`+0.00%`

---

### 4. 成交量格式化 ✅
**問題**: 數字太長難以閱讀（如 `929886`）

**解決方案**:
```python
"成交量": int(quote['volume'])
```

**顯示格式**:
```python
"成交量": "{:,}"  # 千分位逗號
```

**效果**:
- ✅ 加上千分位逗號
- ✅ 例如：`929,886`、`175,7011`
- ✅ 更容易判斷成交量大小

---

### 5. 成長率格式化 ✅
**問題**: 沒有 % 符號，數據不夠直觀

**解決方案**:
```python
"週成長率": round(growth_1, 2),
"月成長率": round(growth_2, 2),
"雙月成長率": round(growth_3, 2)
```

**顯示格式**:
```python
"週成長率": "{:+.2f}%",
"月成長率": "{:+.2f}%",
"雙月成長率": "{:+.2f}%"
```

**效果**:
- ✅ 所有成長率都帶 % 符號
- ✅ 自動顯示正負號
- ✅ 固定 2 位小數

**範例**:
- 週成長率：`+3.21%`
- 月成長率：`-0.28%`
- 雙月成長率：`+3.40%`

---

### 6. 潛力評分格式化 ✅
**優化**: 統一為 1 位小數

**解決方案**:
```python
"潛力評分": round(potential['score'], 1)
```

**顯示格式**:
```python
"潛力評分": "{:.1f}"  # 例如: 100.0, 95.0
```

**效果**:
- ✅ 所有評分統一 1 位小數
- ✅ 例如：`100.0`、`95.0`、`77.0`

---

### 7. 型態欄位 ✅
**保留顯示**，因為提供重要的技術型態信息

**可能值**:
- `多頭` - 週成長>20% 且 月成長>15%
- `突破` - 週成長>10% 且 當日漲幅>3%
- `弱勢` - 週成長<-10%
- `-` - 無明顯型態

**建議**: 保留此欄位，它能快速判斷股票當前狀態
