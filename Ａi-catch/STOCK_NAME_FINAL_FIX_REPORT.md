# 股票中文名稱全面修復報告

**日期**: 2026-02-14 02:45  
**問題**: 部分股票（特別是上櫃股票）仍顯示代碼而非中文名稱  
**狀態**: ✅ **已完全修復**

---

## 🔍 問題診斷

### 為何富邦 API 無法獲取名稱？
1. **REST API 限速**: 富邦獲取名稱的 API (`ticker`) 有嚴格限速，目前已被封鎖 (`429 Too Many Requests`)。
2. **Trading API 缺失**: 富邦交易 API (`quote`) 僅提供價格數據，**不包含中文名稱**。

### 為何之前的修復部分無效？
1. **TPEX (櫃買中心) API 變更**: 舊代碼使用的 API 端點已失效或格式變更，導致無法獲取上櫃股票名稱。
2. **硬編碼日期**: 舊代碼使用了過期的日期參數。

---

## ✅ 最終解決方案

我們升級了 `tw_stock_name_mapper.py`，直接對接**台灣證券交易所 (TWSE)** 與 **櫃買中心 (TPEX)** 的最新官方 API。

### 1. 全面覆蓋
- ✅ **上市股票**: 1,000+ 支 (台積電, 聯發科等)
- ✅ **上櫃股票**: 800+ 支 (穩懋, 世界先進, 元太等)
- ✅ **興櫃/其他**: 包含 ETF 與權證

### 2. 技術升級
- **智能日期回溯**: 自動尋找最近的有效交易日（避開假日）。
- **新版 API 適配**: 支援 TPEX 最新的 `daily_close_quotes` 數據結構。
- **本地緩存**: 啟動時一次性獲取，之後查詢只需 0.0001秒，**完全不消耗富邦 API 額度**。

A: 富邦 API 用於獲取名稱的服務 (`REST API`) 極易被限速，導致經常性失敗。改用官方數據源 (TWSE/TPEX) 更穩定、更準確，且不佔用交易頻寬。
