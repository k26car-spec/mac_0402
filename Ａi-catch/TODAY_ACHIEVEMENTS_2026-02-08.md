### 1. 分割比例很重要

```python
# ❌ 太小的测试集不稳定

# ✅ 更平衡的分割更可靠

```

### 2. 强正则化对过拟合极度有效

```python
# 对严重过拟合（gap>1.0）
config.l2_reg = 0.02
config.dropout_rate = 0.4
config.recurrent_dropout = 0.25

# 效果: MAE降低90%+
```

### 3. 数据扩增也非常有效

```python
# 样本翻倍 + 噪声
X_aug, y_aug = augment_data(X, y, noise_level=0.003)

# 效果: MAE降低90%+
```
