# 📈 潛力股監控儀表板 - 使用說明

## 🎯 功能特色

### ✅ 整合富邦 API
- **即時行情**: 透過富邦證券 API 獲取即時股票報價
- **歷史數據**: 自動獲取歷史 K 線計算成長率
- **自動更新**: 支援 60 秒自動刷新數據
- **批次查詢**: 一次獲取多支股票數據

### 📊 分析功能
1. **多週期成長率分析**
   - 週成長率 (7天)
   - 月成長率 (30天)
   - 雙月成長率 (60天)

2. **智能型態識別**
   - **多頭**: 週成長>20% 且 月成長>15%
   - **突破**: 週成長>10% 且 當日漲幅>3%
   - **弱勢**: 週成長<-10%

3. **優先度評分**
   - **99分**: 高度關注 (週成長>50% 或 雙月成長>100%)
   - **10分**: 一般監控

### 📋 介面功能
- **雙模式顯示**: 簡潔模式 / 完整模式
- **智能篩選**: 按成長率、優先度篩選
- **視覺化圖表**: 
  - 成長率比較圖
  - 型態分布圖
  - 優先度分析圖
- **自訂監控清單**: 隨時新增股票

## 📖 使用指南

### 1. 側邊欄設定

#### 顯示模式
- **簡潔**: 顯示核心欄位（代號、名稱、優先、現價、漲跌幅、週成長率、型態）
- **完整**: 顯示所有欄位（包含成交量、各期成長率、更新時間）

#### 日期範圍
- 設定回測的歷史數據範圍
- 預設：過去 30 天

#### 策略選擇
- 價格波動率突破
- 均線糾結
- 量增價揚
- 動量突破
- 支撐反彈

#### 篩選條件
- **最低成長率**: 設定週成長率的最低門檻
- **僅顯示高優先度**: 只顯示評分 ≥99 的股票

### 2. 新增股票

1. 在側邊欄找到「➕ 新增股票」區塊
2. 輸入股票代號（例如：2330）
3. 點擊「新增到監控清單」
4. 系統會自動獲取該股票的即時數據

### 3. 數據解讀

#### 統計卡片
- **監控股票**: 當前符合篩選條件的股票總數
- **高優先度**: 評分 ≥99 的股票數量
- **多頭型態**: 符合多頭定義的股票數量
- **平均週成長**: 所有股票的平均週成長率

#### 表格顏色
- **黃底**: 高優先度股票（優先=99）
- **綠色數字**: 正成長（▲）
- **紅色數字**: 負成長（▼）
- **灰色數字**: 持平

### 4. 視覺化分析

#### Tab 1: 成長率比較
- 橫向比較各股票的週、月、雙月成長率
- 快速找出成長最強的股票

#### Tab 2: 型態分布
- 查看各種型態的股票數量分布
- 了解整體市場趨勢

#### Tab 3: 優先度分析
- 查看高/低優先度股票的分布
- 決定資源配置

## ⚠️ 常見問題

### Q: 顯示「獲取數據失敗」
**A**: 檢查以下項目：
1. 富邦 API 憑證是否正確設定在 `.env`
2. 網路連線是否正常
3. 是否超出 API 呼叫限制

### Q: 數據不更新
**A**: 
1. 檢查右上角「🔄 啟用自動重新整理」是否勾選
2. 手動點擊 Streamlit 右上角的 ⋮ 選單 → Rerun
3. 清除快取：Settings → Clear Cache

### Q: 股票代號無法新增
**A**:
1. 確認股票代號正確（不含 .TW 或 .TWO）
2. 確認該股票在富邦 API 中有數據
3. 檢查是否已在清單中

### Q: 歷史數據不足
**A**:
- 新上市股票可能缺少足夠的歷史數據
- 成長率計算需要至少 2 個交易日的數據
- 考慮使用較短的計算週期
