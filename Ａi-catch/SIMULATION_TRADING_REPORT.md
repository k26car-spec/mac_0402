# 双重停损机制
stop_loss_strategies = {

#### 3. 趋势过滤器

**现有问题**:
- 50%逆势买入
- 25%追高买入

**改进方案**:
```python
# 严格趋势过滤
def check_trend(data):
    # 必要条件
    if not (MA5 > MA20):
        return False, "下降趋势，拒绝进场"
    
    # 价格位置
    deviation = (close - MA5) / MA5 * 100
    if deviation > 5:
        return False, "偏离MA5超过5%，追高风险"
    if deviation < -3:
        return False, "偏离MA5低于-3%，弱势"
    
    return True, "趋势良好"
```
