每個顯示：
- 信心度百分比
- 進度條
- 期間標籤

#### E. 關鍵證據展示 ✨
- 顯示評分 ≥ 80 的專家
- 展示具體證據
- 綠色背景卡片

## 📊 數據結構

### 主力分析數據
```typescript
{
  symbol: "2330",
  confidence: 0.857,          // 綜合信心度
  action: "entry",            // 動作：entry/exit/hold
  actionReason: "...",        // 動作原因
  timeframe: {
    daily: 0.85,             // 日線信心度
    weekly: 0.78,            // 週線信心度
    monthly: 0.72            // 月線信心度
  },
  riskLevel: "medium",        // 風險等級
  recommendation: "...",      // 建議
  signals: [
    {
      name: "大單分析",
      score: 0.92,            // 評分 0-1
      weight: 0.15,           // 權重 0-1
      status: "bullish",      // 看多/看空/中性
      evidence: [...],        // 證據列表
      confidence: 0.92        // 該專家的信心度
    },
    // ... 15 位專家
  ]
}
```
