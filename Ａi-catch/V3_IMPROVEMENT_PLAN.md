## 🎯 v3.0 核心目標

升級為**機構級主力偵測系統**，新增三大專業特徵：

### ⭐ 新增功能

1. **籌碼集中度追蹤** (12% 權重)
   - 追蹤籌碼從散戶流向主力
   - 領先指標，比量價更早發現主力
   
2. **連續進場天數** (10% 權重)  
   - 過濾假突破
   - 確認主力持續性
   
3. **時段動能分析** (8% 權重)
   - 分析盤中不同時段的主力行為
   - 識別操作模式（早盤吸、尾盤拉）

## 📊 數據來源確認

### 富邦 API 能力評估

**目前已整合:**
- ✅ Yahoo Finance API (主要)
- ✅ 富邦證券 API (備用)
- ✅ 股票名稱自動獲取

**v3.0 需要的數據:**
1. **籌碼集中度** - 需要歷史成交價量分佈
2. **連續進場** - 需要歷史警報記錄（本地資料庫）
3. **時段分析** - 需要日內分時數據

**數據來源決策:**
- 籌碼集中度: 用日線數據 + 統計方法**推估**
- 連續進場: 用本地 SQLite 資料庫記錄
- 時段分析: 用開高低收數據**模擬**盤中力道
