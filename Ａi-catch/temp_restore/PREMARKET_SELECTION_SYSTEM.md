# 開盤前5分鐘精準選股系統 - 完整說明文件

## 📖 系統概述

**開盤前5分鐘精準選股系統** 是一個全自動化、多維度的股票篩選與決策系統，旨在通過整合國際市場數據、法人籌碼、技術指標和即時新聞，在每個交易日開盤前的關鍵5分鐘內，為交易者提供高信心度的精選股票標的。

### 🎯 核心價值

- **提高勝率**: 透過系統化、多條件交叉驗證，提升選股準確度
- **節省時間**: 自動化分析取代人工做功課，解放交易者時間
- **紀律執行**: 提供明確的進場價、目標價、停損價，避免情緒化交易
- **風險控管**: 嚴格的停損機制和倉位管理建議

## ⏰ 完整時間軸與工作流程

系統分為 **4大階段**，從前一天晚上開始準備，直到開盤後執行決策。

### 📅 階段一：前一天晚上 (21:00-23:00)

**目標**: 分析美股表現、國際新聞、法人動向，提前預判明天台股開盤走勢

#### 1.1 美股盤勢分析

**監控指標**:
- **三大指數**: 那斯達克、道瓊、S&P 500

  - Apple (AAPL) → 影響台系供應鏈
  - AMD → 影響台積電
  - Tesla (TSLA) → 影響電動車概念股
- **恐慌指標**: VIX 恐慌指數、Fear & Greed Index
- **期貨市場**: 台指期夜盤、日經期貨

**判斷邏輯**:
```python
if nasdaq > +1.5% and nvidia > +3%:
    預測 = "強勢開高"
    焦點類股 = ["AI概念股", "半導體", "IC設計"]

elif dow < -1.0% and vix > 20:
```

**新聞來源**:

**關鍵字監控**:
- **正面**: 突破、訂單、法說會、配息、併購、新品
- **負面**: 下修、裁員、虧損、調查、禁令
- **政策**: Fed、利率、關稅、補助、法規

**實戰案例**:

### 🔥 階段三：開盤前5分鐘決戰 (08:55-09:00)

**目標**: 整合所有分析，產生 Top 5 精選標的

#### 3.1 綜合評分系統

**權重設定**:
- 美股影響: 40%
- 法人動向: 30%
- 技術面: 20%
- 即時新聞: 10%

**評分邏輯**:
```python
total_score = (
    us_impact_score * 0.4 + 
    institutional_score * 0.3 + 
    technical_score * 0.2 + 
    news_score * 0.1
)

# 必須至少符合 3 個條件才進場
if conditions_met >= 3:
    加入精選名單
```

#### 3.2 技術面篩選條件

**5大技術訊號**:
1. **突破型態**: 突破20日均線 + 量增 (權重30分)
2. **多頭排列**: MA5 > MA10 > MA20 > MA60 (權重25分)
3. **黃金交叉**: MA5 上穿 MA20 (權重20分)
4. **RSI強勢**: 50 < RSI < 70 (權重15分)
5. **MACD多頭**: MACD > Signal 且柱狀圖放大 (權重10分)

**篩選標準**: 總分 >= 60 分

### 🚀 階段四：開盤後執行 (09:00-09:05)

**目標**: 根據開盤價與量能，做出最終執行決策

#### 4.1 開盤型態判斷

```python
opening_gap = (opening_price - yesterday_close) / yesterday_close

if opening_gap > 1%:
    # 跳空開高
    策略 = "觀察，等回測支撐再進"
    進場點 = opening_price * 0.995  # -0.5%

elif -0.5% < opening_gap < 0.5%:
    # 平開
    策略 = "立即買進"
    緊急度 = "高"

else:
    # 跳空開低
    策略 = "放棄"
    原因 = "開低代表多頭力道不足"
```

#### 4.2 量能確認 (09:00-09:05)

**觀察前5分鐘成交量**:
```python
volume_ratio = first_5min_volume / avg_first_5min

if volume_ratio > 2.0:
    決策 = "立即進場！"  # 量能爆增
elif volume_ratio > 1.5:
    決策 = "正常進場"
else:
    決策 = "放棄"  # 量能不足
```
