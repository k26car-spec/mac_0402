## 🐛 故障排除

### 問題: 無法訪問網頁

**解決**:
1. 確認平台已啟動
2. 檢查端口是否正確（8080）
3. 清除瀏覽器快取

### 問題: 端口被占用

**解決**:
```bash
# 查看占用情況
lsof -i :8080

# 停止占用進程
kill [PID]

# 或改用其他端口
# 編輯 dashboard.py 最後一行
```

### 問題: 啟動失敗

**解決**:
```bash
# 1. 停止所有舊進程
pkill -f dashboard.py

# 2. 檢查 Flask 是否安裝
pip3 list | grep Flask

# 3. 重新安裝（如需要）
pip3 install flask

# 4. 再次啟動
./start_dashboard.sh
```
