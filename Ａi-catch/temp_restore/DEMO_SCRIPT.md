### 第四部分：实时演示（3分钟）

**步骤1: 启动系统**
```bash
./start_all.sh
```
> "系统支持一键启动，3秒内完成初始化。"

**步骤2: 查看API文档**
```bash
open http://localhost:8000/api/docs
```
> "系统自动生成完整的Swagger文档，所有API都可以在线测试。"

**步骤3: 查看专家列表**
```bash
curl http://localhost:8000/api/analysis/experts
```
> "当前系统有9个专家就绪，可以随时进行分析。"

**演示画面**:
- API文档页面
- 专家列表JSON输出

**步骤4: 运行AI分析（重点）**
```bash
curl -X POST "http://localhost:8000/api/analysis/mainforce?symbol=2330"
```

> "现在我们对台积电(2330)进行实时分析。9个专家会并行工作，在100毫秒内给出结果。"

**结果解读**:
- 综合信号: BUY/SELL/HOLD
- 信号强度: 65%
- 置信度: 71%
- 专家共识: 6买 / 1卖 / 2持

> "可以看到，9个专家中有6个给出买入信号，综合置信度达到71%，这是一个相对可靠的买入信号。"

**演示画面**:
- 分析结果JSON
- 突出显示关键字段
