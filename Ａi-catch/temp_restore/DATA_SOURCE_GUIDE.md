## 📡 API 端點

### 1. 券商進出數據

**端點**: `GET /api/stock-selector/broker-flow/fubon-xindan/top-stocks`

**參數**:
- `top_n`: 前 N 名（預設 20）

**返回**:
```json
[
  {
    "stock_code": "2330",
    "stock_name": "台積電",
    "buy_count": 1500,
    "sell_count": 800,
    "net_count": 700
  }
]
```

---

### 2. 執行選股分析

**端點**: `POST /api/stock-selector/analyze/batch`

**請求**:
```json
{
  "auto_fetch_from_broker": true,
  "broker_code": "9600",
  "top_n": 20
}
```

**返回**:
```json
[
  {
    "股票代碼": "2330",
    "綜合評分": 85.5,
    "評級": "A+",
    "建議動作": "強力買入",
    "目標價": 1150,
    "停損價": 950,
    "建議倉位(%)": 12,
    "風險等級": "medium",
    "基本面分數": 88,
    "技術面分數": 82,
    "籌碼面分數": 86
  }
]
```
